# ShoeHub - E-commerce Shoe Website

A modern, responsive e-commerce website for shoes built with HTML, CSS, and JavaScript.

## Features

### 🏠 Home Page
- Hero banner with call-to-action
- Featured categories (Sneakers, Formal Shoes, Sports Shoes, Sandals)
- Best sellers section
- New arrivals
- Why choose us (Free shipping, Easy returns, Premium quality)
- Customer reviews
- Newsletter signup

### 🛍️ Shop Page
- Product filtering by price, size, color, category
- Sorting options (popularity, price, newest)
- Product grid with images and details

### 👨/👩/👶 Category Pages
- Men's Shoes
- Women's Shoes
- Kids' Shoes
- Each with dedicated product grids

### 🎯 Collections Page
- Summer Collection
- Running Shoes
- Premium Leather
- Curated product collections

### 🧾 About Us
- Brand story and mission
- Company values
- Founder information

### ✍️ Blog
- Shoe care tips
- Fashion trends
- Product guides

### 📞 Contact Page
- Contact form
- Business information
- Embedded map

### 🛒 Shopping Cart
- Add/remove items
- Quantity management
- Price calculations
- Persistent storage

### 👤 User Account
- Login/Register forms
- Account management

## Technical Features

- **Responsive Design**: Mobile-first approach, works on all devices
- **Modern UI**: Clean design with smooth animations
- **Product Management**: Dynamic product loading and filtering
- **Cart Functionality**: Local storage for cart persistence
- **Form Validation**: Client-side validation for forms
- **SEO Friendly**: Proper HTML structure and meta tags

- **HTML5**: Semantic markup

- **JavaScript**: ES6+ features, DOM manipulation
- **Local Storage**: For cart persistence
